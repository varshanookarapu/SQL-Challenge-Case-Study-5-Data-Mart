# 3. Before & After Analysis

This technique is usually used when we inspect an important event and want to inspect the impact before and after a certain point in time.
Taking the week_date value of 2020-06-15 as the baseline week where the Data Mart sustainable packaging changes came into effect.
We would include all week_date values for 2020-06-15 as the start of the period after the change and the previous week_date values would be before
Using this analysis approach - answer the following questions:

**Question 1:** What is the total sales for the 4 weeks before and after 2020-06-15? What is the growth or reduction rate in actual values and percentage of sales?

solved this question in two parts , since the first part is to get the total sales for 4 weeks before and after 2020-06-2015 , I created two ctes before and after , 


```sql
--This CTE gives the weekly breakdown of total sales for 4 weeks before 2020-06-2015
with before_sales AS
(
SELECT  week_date,sum(sales) as four_weeks_before_total_sales 
from cleaned_weekly_sales
WHERE week_date BETWEEN DATE'2020-06-15' - INTERVAL '4 weeks'  AND  DATE'2020-06-15' - INTERVAL '1 weeks'
GROUP BY week_date
)
SELECT * FROM before_sales 
```

To address the second part of this question which is the What is the growth or reduction rate in actual values and percentage of sales

```sql
WITH sales AS
(
SELECT
SUM(CASE WHEN (week_date BETWEEN DATE'2020-06-15' - INTERVAL '4 weeks'  AND  DATE'2020-06-15' - INTERVAL '1 weeks')      
THEN sales 
END ) AS  four_weeks_before_total_sales ,
     
 SUM( CASE  WHEN (week_date BETWEEN  DATE'2020-06-15'  AND DATE'2020-06-15' +  INTERVAL '4 weeks') THEN
     sales END ) AS  four_weeks_after_total_sales
FROM cleaned_weekly_sales

)

SELECT ( four_weeks_after_total_sales -four_weeks_before_total_sales  ) AS change_in_sales,
ROUND((100*( four_weeks_after_total_sales -four_weeks_before_total_sales  ) /NULLIF(four_weeks_before_total_sales,0)),2) :: DECIMAL AS 
percentage_change_in_sales
FROM sales ; 

```
Conclusion :  after observing the acutal change in sales is positive , since they have introduces the sustainable packaging there has been a 23% of growth in sales within the 4 weeks period of implementation. Showing a postive  impact on the customer purchase behaviour
