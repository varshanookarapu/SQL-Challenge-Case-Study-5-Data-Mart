```sql
week_date,
TO_CHAR(week_date, 'Day') as day
FROM cleaned_weekly_sales

ORDER BY week_date
```

**Question 2:** What range of week numbers are missing from the dataset?

```sql

-- Generate Series ( generates the week numbers from 1 to 52 ) and Except function - it returns the values that are not in the weeknumbers from the cleaned weekly sales table

SELECT DISTINCT week_number FROM cleaned_weekly_sales  ORDER BY week_number
```

**Question 3:** How many total transactions were there for each year in the dataset?

```sql
SELECT calendar_year,COUNT(*) 
FROM cleaned_weekly_sales  
GROUP BY calendar_year
ORDER BY calendar_year
```

**Question 4:** What is the total sales for each region for each month?

```sql

SELECT  region,month_number,TO_CHAR(week_date,'Month') as month , SUM(sales) as total_sales FROM 
cleaned_weekly_sales
GROUP BY region,month,month_number
ORDER BY region,month_number
```
