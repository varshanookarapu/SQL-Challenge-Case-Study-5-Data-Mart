**Question 6:** What is the percentage of sales for Retail vs Shopify for each month?
```sql
SELECT platform, month_number as month ,  
ROUND( 100.0 * SUM(sales) 
       / SUM(SUM(sales)) OVER (PARTITION BY month_number),2) AS percentage_of_sales
FROM cleaned_weekly_sales
GROUP BY platform, month_number 
ORDER BY platform, month_number
```

**Question 8:** Which age_band and demographic values contribute the most to Retail sales?

```sql
WITH cte AS
(

SELECT platform,demographic,age_band,SUM(sales) retail_sales,
  
  -- I was a big stuck at this part where I was confused on how to get the  max retails sales for every demographic but this is the sceario where we use the Ranking concept. 
  RANK() OVER(PARTITION BY demographic ORDER BY SUM(sales) DESC) as rank
  FROM cleaned_weekly_sales

WHERE platform= 'Retail'
GROUP BY demographic,age_band,platform
)

SELECT * from cte WHERE rank=1 
ORDER BY retail_sales DESC
```
Unknown Demographic and Age band contribute most to the retail sales followed by  Familes - Retires and Couples-Retirees

SELECT platform,calendar_year, ROUND(AVG(avg_transaction),2) as transaction_size ,ROUND( SUM(sales)/SUM(transactions),2) as transaction_size_wo_avg_transaction
