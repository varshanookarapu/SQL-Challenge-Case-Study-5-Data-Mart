## Question 1 :   What day of the week is used for each week_date value?

```sql
-- Monday
SELECT 
week_date,
TO_CHAR(week_date, 'Day') as day
FROM cleaned_weekly_sales 
GROUP BY week_date
ORDER BY week_date

-- Had to check if there were any other days of the week hence the above query. 

SELECT 
DISTINCT TO_CHAR(week_date, 'Day') as day
FROM cleaned_weekly_sales 


```

## Question 2 : What range of week numbers are missing from the dataset?

```sql

-- Generate Series ( generates the week numbers from 1 to 52 ) and Except function - it returns the values that are not in the weeknumbers from the cleaned weekly sales table 

SELECT generate_series(1,52) as week_number
EXCEPT
SELECT DISTINCT week_number FROM cleaned_weekly_sales  ORDER BY week_number

```
